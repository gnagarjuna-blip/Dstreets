How to Use This Tool

  1. Enter initial numbers: Type comma-separated numbers (0-36) in the input field and click "Load Initial Numbers" OR click individual number buttons.
  2. Results update automatically: As you click numbers, the tool automatically calculates and displays the 3 double streets missing the longest.
  3. Double streets: DS1: 1-6, DS2: 7-12, DS3: 13-18, DS4: 19-24, DS5: 25-30, DS6: 31-36. Number 0 doesn't belong to any double street.
  4. Clear history: Use "Clear All" to reset the history and start over.
  5. Missing double streets: Shows the 3 double streets that haven't appeared for the longest time in your number history.
