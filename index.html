How to Use This Tool

  1. Enter initial numbers: You can either type comma-separated numbers (0-36) in the input field and click "Load Initial Numbers" OR click individual number buttons.
  2. Add more numbers: Continue clicking number buttons to add them to the history.
  3. Calculate missing double streets: Click "Calculate Missing Double Streets" to identify the 3 double streets that have been missing the longest.
  4. Clear history: Use "Clear All" to reset the history and start over.
  5. Double streets explained: DS1: 1-6, DS2: 7-12, DS3: 13-18, DS4: 19-24, DS5: 25-30, DS6: 31-36. Number 0 doesn't belong to any double street.
